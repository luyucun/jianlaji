# 核心玩法循环

## 一句话概括

玩家在**单局捡垃圾**中开盲盒式挖掘、变现、成长，单局结束后，稀有物品会作为**永久解锁**留在自己的家园里展示——形成"局内爽感 + 局外长期收藏目标"的双循环。

---

## 整体结构：两层循环

```
        ┌─────────────────────────────┐
        │         局外循环（家园）      │
        │   家园展示 → 解锁目标 → 开新局 │
        └───────────────┬─────────────┘
                         │ 开一局
                         ▼
        ┌─────────────────────────────┐
        │         局内循环（对局）      │
        │  翻找 → 发现 → 变现/成长 → 结算│
        └───────────────┬─────────────┘
                         │ 局结束，稀有物解锁
                         ▼
                 回到家园，置物架点亮
```

局内是**短周期的情绪爽感**（开盲盒、随机惊喜、即时反馈），局外是**长周期的目标感**（图鉴收藏、家园成长）。两者通过"稀有物首次获得=解锁"这一个动作串联起来。

---

## 局内循环：单局对局流程

1. **开局**：玩家选择 1 人局 / 2 人局 / 4 人局，匹配后进入对应的垃圾场景。
2. **翻找**：在场景中持续挖掘、翻动垃圾堆，维持"下一件是什么"的期待感。
3. **掉落判定**：随机产出物品，按稀有度分层——普通垃圾、可出售物、稀有物品。
4. **局内决策**：
   - 普通/可出售物 → 当场卖掉换钱
   - 钱 → 在局内用于提升自己的各项能力维度（比如翻找速度、挖掘范围、移动等，具体维度后续再定）
   - 稀有物品 → 除了能卖钱变现，**若是本局或历史首次获得，则触发解锁**
5. **结算**：单局结束（时间到/完成目标等，具体规则后续再定），本局的钱和能力提升不带出对局，但**首次解锁的稀有物品记录为永久成就**。

局内的核心体验是"低压力、无战斗、持续小惊喜"，玩家在一局的时间尺度内感受到成长（钱→能力提升→翻找更顺畅→发现更多）。

---

## 局外循环：家园与长期目标

1. **家园**：每个玩家有自己的基础 home，摆放着卡片展示柜和置物架。
2. **展示解锁**：只要某个稀有物品被首次获得过，回到家园后对应的展示柜/置物架就会亮起，永久陈列这件物品。
3. **驱动效果**：家园里空着的柜子/架子，天然构成"我还没有捡到过什么"的收藏缺口，驱动玩家再开一局去补全。
4. **循环起点**：玩家看着自己的家园，决定再开一局（1/2/4人），进入下一次局内循环。

局外的核心体验是"我的家在慢慢被填满"，是纯正向的收藏成就感，没有惩罚、没有消耗。

---

## 两层循环的关系

| | 局内循环 | 局外循环 |
|---|---|---|
| 时间尺度 | 单局（分钟级） | 长期（跨局） |
| 驱动力 | 随机发现的即时惊喜、变现成长 | 收藏图鉴的目标感、家园成长 |
| 情绪曲线 | 平静翻找 → 意外发现 → 变现/成长 → 满足感 | 家园逐渐被填满 → 期待还缺的那几件 |
| 产出留存 | 钱和局内能力提升不保留 | 首次解锁的稀有物永久保留并展示 |
| 连接方式 | 局结束 → 触发解锁判定 → 进入局外 | 看到家园缺口 → 开新局 → 进入局内 |

这套双循环让玩家既能在每一局里获得独立完整的爽感（不依赖长期养成才好玩），又能通过家园的可视化收藏驱动长期留存（每一局都在为家园添砖加瓦）。
